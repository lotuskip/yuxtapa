Planned Features

On this page are listed a number of ideas that might be implemented into some future release of yuxtapa. You are welcome to contribute new ideas or comment on these. Nothing major is going to happen, though.

  1. trappers can also disarm traps
  2. more mining abilities (these make sense but might make miners too 'all-purpose'):
    • breaking windows just like walls (and even faster)
    • destroying closed doors completely by digging into them
    • destroying static torches similarly
    • clearing rough terrain
  3. to counter-balance that: make some walls undiggable (would look like regular walls but just not break)?
  4. fireballs light carried torches (a minor detail, but could be a nice touch?)
  5. acid traps (much like boobytrap, but doesn't vanish once triggered)
  6. illusory walls (walk-through but not see-through tiles)
  7. more statistics:
    • number of team kills
    • stat kills according to the method of killing (see ways to die)
    • suggestions?
  8. some kind of mind's eye type ability that would leave the PC standing vulnerable, but would allow moving the viewpoint quickly just like a spectator's (perhaps a new class for this?)
  9. possibility to save the currently played map with a given name into a file and load a previously saved map to play (if the game happens to generate an awesome map, you wouldn't lose it; e.g. "!storemap coolmap", "!nextmap coolmap")
  10. make chatting more user-friendly:
    • possibility to scroll previously entered strings with up/down arrow keys
    • smarter aliasing in the config (suggestions?)
  11. should spectators see not only the view of the followed player but also their PC info and the messages that are currently sent only to them?
  12. an actual AI for the bots
  13. new game modes?
  14. optional sound effects in the client end? (in addition to the visual '!'s you could actually hear a sound)
