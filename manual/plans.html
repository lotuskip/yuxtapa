Planned Features

On this page are listed a number of ideas that might be implemented into some future release of yuxtapa. You are welcome to contribute new ideas or comment on these. Nothing major is going to happen, though.

Game features&rules related:

  1. more mining abilities:
    • destroying closed doors by digging into them
    • turning floor into rough terrain (not always succesful, a chance of creating a chasm in dungeons, ...)
  2. illusory walls (walk-through but not see-through tiles)
  3. maybe take the facing variable into account more often (like divide attacks to frontal/flank/rear and adjust DV correspondingly)? This would be something you could turn on in the server config; players who dislike it could run 'old-school' servers where it's turned off.
  4. new game modes; got suggestions?
Metagaming&interface related:
  1. smarter, more advanced chat aliasing (suggestions?)
  2. make the static entities (flags, torches, boulders, portals etc.) part of the stored maps (a nice idea but appears to demand a lot of rethinking and work — this was never planned!)
  3. an actual AI for the bots
  4. optional sound effects in the client end? (in addition to the visual '!'s you could actually hear a sound)
