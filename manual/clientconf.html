Client configuration

A template configuration file is contained in the yuxtapa tarball; it is under the directory 'yuxtapa/tmplates', and named 'client.conf'. AUR users will find this file in '/usr/share/yuxtapa/' after installing.

This configuration file, when in use, typically resides under

/home/username/.config/yuxtapa/
It is looked for in in '$HOME/.config/yuxtapa/'. You should create this directory and copy the template file 'client.conf' there.

You can edit the file with your favourite text editor. The provided template should be commented to be self-explanatory, see it.
