Client configuration

The yuxtapa client configuration, like the Server configuration, is typically in the file

/home/username/.config/yuxtapa/client.conf
It is first looked for in '$XGD_CONFIG_HOME/yuxtapa/', and secondarily, if XGD_CONFIG_HOME is not defined, in '$HOME/.config/yuxtapa/'. You should create this directory and copy the file 'tmplates/client.conf' there.

You can edit the file with your favourite text editor. The provided template file should be commented to be self-explanatory, see it.
