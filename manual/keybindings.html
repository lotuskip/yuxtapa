Key bindings

This page lists the default key bindings and explains their functions. On remapping the keys, see Client configuration. Note that lower and upper case matters, c really has to be lower case, X really has to be upper case (shift+x).

You can view the current mapping of the keys ingame with the '?' key.

  • number keys 1-4, 6-9 move in the respective direction; 1 bottom left, 2 down, etc.: 
    7 8 9

4 @ 6

1 2 3
    • obviously the idea is to use the numpad with numlock turned on
    • you can also use the arrow keys (but they don't work in diagonal directions)
    • when playing an Archer, the same keys are used to move the aimer
    • whenever in this manual you are instructed to enter a direction, this means to enter the key that normally makes you move into that direction
    • see also Movement, especially regarding the walking mode
  • 5 stop without leaving walking mode. Archers and healers also needs this
  • space special action; use class ability (see Class overview); or switch followed player if spectating
  • p switch to follow previous player if spectating (loops in opposite direction to 'space' switch)
  • C, s chat/command, say/shout; see Chatting and shouting
  • c close a door; see Objects in the game
  • X suicide; kill your player character (useful on occasion)
  • T trigger a trap (while standing on it); see Traps
  • u use torch (light or extinguish); see Vision and hearing and PC torch
  • w toggle the walking mode (see Movement)
  • t toggle titles (see Vision and hearing)
  • o toggle reacting to damage (should a shout be generated if you take damage or not)
  • +, - scroll the chat buffer
  • l limbo toggle (see Limbo and spawning)
  • Q quit; disconnect and end the client program
