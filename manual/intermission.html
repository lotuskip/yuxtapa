Intermission

A yuxtapa server alternates between two phases: the game phase, during which the actual playing happends, and an intermission. After each intermission, the played map, and possibly the objective, are changed.

During the intermission, the server shows a condensed image of the next map and a list of the current players, and the players can take a break. The intermission's length is determined in the Server configuration. Chatting, switching team and changing class are all possible during the intermission, just like in the game phase.
