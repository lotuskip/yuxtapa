Intermission

A yuxtapa server alternates between two phases: the game phase, during which the actual playing happends, and an intermission. After each intermission, the played map, and possibly the objective, are changed.

During the intermission, the server shows a condensed image of the next map and a list of the current players, and the players can take a break. The length of the intermission is determined in the Server configuration. Chatting, switching team and changing class are all possible during the intermission, just like in the game phase.

Turning the titles (see Vision and hearing) on during the intermission will view the objective of the next map (see Game types); the objective viewn in the normal place in the upper left corner is the objective (and result) of the previous map.

The update animation does not run during intermission! So worry not, your connection to the server is (probably) fine.
